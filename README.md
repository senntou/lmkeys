# lmkeys

LMDBデータベースのキー一覧をターミナル上で確認するためのTUIツールです。

## 機能

- LMDBデータベースのキーを一覧表示
- ページネーション機能（デフォルト30件/ページ）
- h/lキー（またはカーソルキー）でページ移動
- キーと値の型情報を表示
- UTF-8デコード可能なキーは文字列として、不可能なキーは16進数で表示
- 読み取り専用モードで安全にアクセス

## インストール

### 開発環境でのインストール

```bash
# リポジトリのクローンまたは移動
cd /path/to/lmkeys

# 開発モードでインストール
pip install -e .
```

### 通常のインストール

```bash
pip install .
```

## 使い方

基本的な使い方:

```bash
lmkeys /path/to/lmdb/directory
```

ページサイズを指定:

```bash
lmkeys /path/to/lmdb/directory --page-size 50
```

## キー操作

- `h` または `←`: 前のページへ
- `l` または `→`: 次のページへ
- `q`: 終了

## 値の型表示

値の型は以下のように判定されます:

- **str**: UTF-8でデコード可能で印字可能な文字列
- **int/bytes**: 1, 2, 4, 8バイトの整数の可能性があるデータ
- **bytes**: その他のバイナリデータ

各値には型名とバイト数が表示されます（例: `str (256 bytes)`）

## 技術仕様

- **Python**: 3.8以上
- **依存ライブラリ**:
  - lmdb >= 1.4.0
  - textual >= 0.47.0
